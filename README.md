# RDFViz

RDFViz es un visualizador de datos RDF escrito en PHP.

# Instalación y uso

  - Clona o descarga este repositorio.
  - Coloca los archivos dentro de la carpeta HTDOCS en el servidor local que estés usando: xampp/mamp/wamp.
  - Cargar cualquiera de los archivos HTML dependiendo de la visualización deseada. Existe el tipo tabla de datos y gráfico de barras.
  - En el campo ENDPOINT ingresa la url de la fuente de datos.
  - Ingresa una consulta  o solo pulsa consultar para ver los resultados de la consulta predefinida.


### Dependencias Usadas

Este proyecto usa las siguientes librerías:

* [EasyRDF](http://www.easyrdf.org/) - A PHP library designed to make it easy to consume and produce RDF.
* [D3.JS](https://d3js.org/) -  is a JavaScript library for manipulating documents based on data.
